# Slide 11. Codex 데모 흐름

오늘 데모는 전체 구현이 아니라 안전한 요청 방식만 보여줍니다.

## 보여줄 것

1. 현재 프로젝트 상태 확인 요청
2. `/about`, `/projects`, `/posts` 계획 받기
3. Layout과 Navigation 계획 받기
4. 파일 수정 승인하기
5. `npm run dev`로 확인하기
6. Browser 또는 일반 브라우저로 페이지 이동 확인하기
7. `npm run build`로 검증하기
8. commit 메시지 추천 받기

---

## 오늘도 먼저 할 말

```text
아직 파일을 수정하지 말고, 먼저 계획을 설명해줘.
내가 승인하면 진행해줘.
```
