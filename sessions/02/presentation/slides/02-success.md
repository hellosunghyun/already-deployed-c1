# Slide 02. 오늘의 성공 기준

오늘 성공 기준은 3단계입니다.

## 최상

- Home / About / Projects / Posts 페이지가 있음
- 모든 페이지에 같은 Navigation이 있음
- Projects 카드가 3개 이상 있음
- Posts 샘플 글 목록이 있음
- `npm run build` 성공
- GitHub Pages에 반영됨

## 표준

- 로컬에서 네 페이지가 열림
- Layout과 Navigation이 적용됨
- build 성공 또는 실패 로그 확보

## 최소

- 현재 프로젝트 상태를 확인함
- 어디서 막혔는지 기록함
- 다음 질문을 만들 수 있음
