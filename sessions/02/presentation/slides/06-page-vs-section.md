# Slide 06. 페이지와 섹션

오늘 가장 헷갈릴 수 있는 단어입니다.

```text
페이지 = 주소가 따로 있는 화면
섹션 = 한 페이지 안의 구역
```

예:

```text
/about = 페이지
Home 안의 "관심 분야" = 섹션
```

---

## 왜 구분해야 하나요?

AI에게 요청할 때 아래 두 문장은 결과가 다릅니다.

```text
About 페이지를 만들어줘.
```

```text
Home 페이지 안에 About 섹션을 만들어줘.
```
