# Slide 09. Component와 데이터

Projects 페이지에는 비슷한 카드가 여러 개 나옵니다.

```text
프로젝트 이름
한 줄 설명
내 역할
사용 도구
결과 또는 배운 점
```

이런 반복 화면은 Component로 만들기 좋습니다.

---

## 데이터와 화면을 나누기

```text
프로젝트 데이터 = 무엇을 보여줄지
ProjectCard Component = 어떻게 보여줄지
```

이렇게 나누면 프로젝트가 3개에서 10개로 늘어나도 추가하기 쉽습니다.
