# Slide 12. 이제 워크북으로 갑니다

오늘 워크북은 아래 순서로 진행합니다.

1. 1회차 결과 확인
2. 현재 프로젝트 상태 점검
3. 페이지 구조 만들기
4. Layout과 Navigation 만들기
5. Home / About / Projects / Posts 채우기
6. 브라우저와 build로 확인
7. commit / push / deploy
8. AGENTS.md를 현재 구조에 맞게 갱신
9. 다음 세션 전 홈페이지 레퍼런스 찾기

---

## 실습 중 규칙

모르면 멈추지 말고, 아래 정보를 담아 AI에게 질문합니다.

- 지금 워크북 단계
- 현재 프로젝트 상태
- 방금 실행한 명령어
- 에러 메시지
- 원하는 결과
