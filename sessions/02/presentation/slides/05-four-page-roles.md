# Slide 05. Home / About / Projects / Posts

각 페이지는 역할이 다릅니다.

| 페이지 | 역할 |
|---|---|
| Home | 첫인상과 안내 |
| About | 내가 어떤 사람인지 설명 |
| Projects | 내가 해본 일의 증거 |
| Posts | 생각과 기록 |

---

## 오늘 만들 최소 내용

- Home: 이름, 한 줄 소개, 주요 링크
- About: 짧은 소개와 관심사
- Projects: 프로젝트 카드 3개
- Posts: 샘플 글 목록 1~2개
