# Slide 03. 1회차 결과는 시작점입니다

1회차 목표는 아래 흐름을 한 번 경험하는 것이었습니다.

```text
만든다
→ 확인한다
→ 저장한다
→ 배포한다
→ 링크로 본다
```

2회차부터는 질문이 바뀝니다.

```text
이 사이트를 처음 본 사람은
무엇을 보고
어디를 눌러야 할까?
```

<details><summary>발표자 노트</summary>

**슬라이드 화면 구성**

- 1회차 흐름: `만든다 → 확인한다 → 저장한다 → 배포한다 → 링크로 본다`
- 2회차 질문: `처음 본 사람은 무엇을 보고 어디를 눌러야 할까?`
- 화면에는 긴 설명보다 두 개의 흐름 비교가 보이면 충분합니다.

**말할 내용**

1회차에서 만든 홈페이지가 아직 어설퍼 보여도 괜찮다고 말합니다.
오늘은 그 결과를 버리고 새로 시작하는 시간이 아닙니다.

AI에게도 이렇게 말하게 합니다.

```text
새로 만들어줘가 아니라,
현재 프로젝트를 유지하면서 구조를 추가해줘.
```
