# Slide 06A. 페이지와 섹션 미니 연습

아래 요청을 보고 **페이지**인지 **섹션**인지 구분해봅니다.

| 요청 | 답 |
|---|---|
| About 화면을 따로 만들고 싶다 | 페이지 |
| Home 안에 관심 분야 소개를 넣고 싶다 | 섹션 |
| Projects 목록을 주소로 열고 싶다 | 페이지 |
| 첫 화면 아래에 연락처 블록을 넣고 싶다 | 섹션 |

---

## AI에게 말할 때 달라지는 점

```text
페이지를 만들어줘 = 주소가 따로 생김
섹션을 만들어줘 = 한 페이지 안에 구역이 생김
```

오늘 만드는 것은 주로 **페이지**입니다.

**슬라이드 화면 구성**

- 4개 예시 표를 그대로 화면에 둡니다.
- 하단에는 `페이지를 만들어줘 = 주소가 따로 생김`, `섹션을 만들어줘 = 한 페이지 안에 구역이 생김`만 남깁니다.

**말할 내용**

여기서는 진행자가 바로 답을 말하지 말고, 참가자에게 먼저 물어봅니다.
2~3분 정도 써도 괜찮습니다.

이 구분이 잡히면 뒤에서 AI에게 요청할 때 훨씬 덜 헷갈립니다.
"About 페이지"와 "Home 안의 About 섹션"이 다르다는 정도만 잡으면 충분합니다.
