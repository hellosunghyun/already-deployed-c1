# Slide 18. Navigation은 안내판입니다

Navigation은 방문자가 이동할 수 있는 길을 보여줍니다.

오늘 필요한 링크:

```text
Home      /
About     /about
Projects  /projects
Posts     /posts
```

Navigation이 있으면 방문자는 주소를 외우지 않아도 됩니다.

<details><summary>슬라이드 모습과 강조점</summary>

**슬라이드 모습**

- 링크 이름과 주소를 짧은 표처럼 보여줍니다.
- `주소를 외우지 않아도 됩니다` 문장을 강조합니다.
- Layout과 연결해서 "모든 페이지에 같은 안내판"이라는 인상을 줍니다.

**강조해야 하는 내용**

- Navigation은 페이지 이동 링크 묶음입니다.
- 모든 페이지에서 같은 Navigation이 보이면 방문자가 길을 잃지 않습니다.
- 링크를 한 곳에서 관리하면 수정이 쉽습니다.

</details>

<details><summary>발표자 노트</summary>

Navigation은 어렵게 말하면 내비게이션이지만, 오늘은 안내판이라고 생각하면 됩니다.
방문자가 Home, About, Projects, Posts로 이동할 수 있게 보여주는 링크 묶음입니다.

주소를 직접 입력하지 않아도 클릭해서 이동할 수 있어야 합니다.
그래서 Layout 안에 Navigation을 넣어 모든 페이지에서 같은 안내판이 보이게 합니다.

</details>
