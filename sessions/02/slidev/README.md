# 2회차 Slidev 발표자료

2회차 Markdown 발표자료를 실제 발표용 Slidev 덱으로 정리한 폴더입니다.

## 실행

```bash
cd sessions/02/slidev
npm install
npm run dev
```

기본 주소는 `http://127.0.0.1:3030`입니다.
Slidev CLI 버전에 맞춰 `npm run dev`가 3030 포트로 바로 실행되도록 설정했습니다.

## 빌드

```bash
npm run build
```

디자인은 `/Users/hellosunghyun/Downloads/PRESENTATION_DESIGN_GUIDE.md`의 색상, 타이포그래피, 카드, 라벨 규칙을 기준으로 만들었습니다.
Slidev의 `style.css` 자동 로드 방식을 사용합니다.
