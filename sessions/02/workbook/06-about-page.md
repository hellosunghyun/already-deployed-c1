# 06. About 페이지 초안 작성

## 목표

About 페이지에 자신을 설명하는 초안을 작성합니다.

## 왜 하는가

About은 긴 이력서가 아닙니다.
처음 보는 사람이 "이 사람이 어떤 배경과 관심사를 가진 사람인지" 이해할 수 있게 하는 페이지입니다.

## 쉬운 설명

About에 들어갈 수 있는 내용은 아래 정도입니다.

- 지금 나를 설명하는 한 문장
- 관심 분야
- 잘하는 것 또는 배우는 중인 것
- 지금 만들고 싶은 것
- 앞으로 더 채워갈 내용

완벽한 문장을 쓰려고 오래 멈추지 않아도 됩니다.
오늘은 초안이면 충분합니다.

## 내 정보 정리

```text
한 줄 소개:
나의 배경:
관심 분야:
지금 배우는 것:
내가 해본 것:
앞으로 하고 싶은 것:
방문자가 나를 어떻게 기억하면 좋을지:
```

## 공통 프롬프트

```text
About 페이지 초안을 작성하고 싶어.

현재 상태:
- Astro 개인 홈페이지가 있다.
- 공통 Layout과 Navigation이 있다.
- /about 페이지는 있지만 내용이 아직 임시 상태다.

내 정보:
- 한 줄 소개: [여기에 입력]
- 배경: [학교, 전공, 일, 동아리, 관심 활동 등 입력. 없으면 비워도 됨]
- 관심 분야: [여기에 입력]
- 지금 배우는 것: [여기에 입력]
- 내가 해본 것: [여기에 입력]
- 앞으로 하고 싶은 것: [여기에 입력]

목표:
- 처음 보는 사람이 나를 이해할 수 있는 About 페이지 초안을 만든다.
- 너무 길지 않게 소제목과 짧은 단락으로 나눈다.
- 과장된 표현보다 구체적인 표현을 사용한다.
- 아직 확정되지 않은 내용은 "배우는 중", "탐색 중"처럼 자연스럽게 표현한다.

문장 톤:
- 자기소개서처럼 딱딱하지 않게
- 하지만 너무 가볍거나 장난스럽지 않게
- "최고의", "혁신적인", "열정적인" 같은 추상적 표현은 줄이기
- 내가 실제로 말할 수 있을 것 같은 문장으로 쓰기

구성 제안:
- 짧은 소개
- 지금 관심 있는 것
- 해본 일 또는 배우는 중인 것
- 앞으로 채워갈 것

제약:
- 새 라이브러리를 추가하지 마.
- About 페이지와 필요한 스타일만 수정해줘.
- Projects나 Posts 내용은 아직 바꾸지 마.
- 관련 없는 파일은 수정하지 마.

작업 방식:
1. 먼저 현재 /about 파일을 확인해줘.
2. About 페이지 구성안을 제안해줘.
3. 내가 제공한 정보 중 부족한 부분이 있으면 임시 문장으로 처리해줘.
4. 아직 수정하지 말고 계획과 문장 초안만 보여줘.
5. 내가 승인하면 파일을 수정해줘.

완료 후 알려줄 것:
1. 변경한 파일 목록
2. About 페이지의 섹션 구성
3. 내가 나중에 직접 바꾸면 좋은 문장
4. 브라우저 확인 주소
```

## 문장 다듬기 요청

초안이 너무 어색하면 이렇게 요청합니다.

```text
About 페이지 문장이 아직 어색해.

아래 기준으로 다시 다듬어줘.
- 더 담백하게
- 더 짧게
- 실제 사람이 자기 홈페이지에 쓸 법하게
- 과장 없이
- 초보자나 처음 방문한 사람도 이해하게

단, 페이지 구조와 코드 구조는 크게 바꾸지 마.
```

## 체크박스

- [ ] About 페이지에 내 소개가 들어갔다.
- [ ] 관심 분야가 보인다.
- [ ] 지금 배우는 것 또는 해본 것이 보인다.
- [ ] 문장이 너무 길지 않다.
- [ ] 과장된 표현이 많지 않다.
- [ ] 처음 보는 사람도 이해할 수 있다.

## Ready Gate

다음 단계로 넘어가기 전에 아래가 되어 있어야 합니다.

- About 페이지가 임시 제목만 있는 상태가 아닙니다.
- 완벽하지 않아도 내 정보가 들어 있습니다.
- 나중에 직접 수정할 문장이 무엇인지 알고 있습니다.

## 힌트

<details>
<summary>Hint 1</summary>

About은 길수록 좋은 페이지가 아닙니다. 3~5개 짧은 단락이면 충분합니다.

</details>

<details>
<summary>Hint 2</summary>

할 말이 없다면 "지금 배우는 것"과 "앞으로 만들고 싶은 것"부터 쓰세요.

</details>

<details>
<summary>Hint 3</summary>

AI에게 내 정보를 너무 적게 주면 추상적인 문장이 나옵니다. 구체적인 단어를 3개 이상 주세요.

</details>
