# 2회차 개인 정보 정리 템플릿

Home과 About 페이지를 만들 때 아래를 먼저 채우면 AI에게 훨씬 좋은 맥락을 줄 수 있습니다.
모든 칸을 완벽히 채우지 않아도 됩니다.

## 기본 정보

```text
이름 또는 닉네임:
한 줄 소개:
지금 내가 하는 일 또는 배우는 것:
관심 분야 1:
관심 분야 2:
관심 분야 3:
```

## About 페이지에 넣을 재료

```text
나의 배경:
요즘 가장 관심 있는 것:
내가 잘하거나 좋아하는 방식:
최근 해본 일:
앞으로 만들고 싶은 것:
방문자가 나를 어떻게 기억하면 좋을지:
```

## 문장 톤 선택

원하는 톤에 체크합니다.

- [ ] 담백한 소개
- [ ] 학생/초보자에게 자연스러운 소개
- [ ] 포트폴리오에 가까운 소개
- [ ] 작업 기록장에 가까운 소개

## AI에게 보낼 때 붙일 문장

```text
아래 정보는 아직 초안이야.
너무 과장하지 말고, 개인 홈페이지에 자연스럽게 들어갈 문장으로 다듬어줘.
부족한 정보는 지어내지 말고 임시 문장 또는 비워둘 자리로 처리해줘.
```
