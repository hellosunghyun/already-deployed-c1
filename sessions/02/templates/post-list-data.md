# 2회차 글 목록 샘플 데이터 템플릿

Posts 페이지에 넣을 샘플 글 목록을 정리하는 템플릿입니다.
오늘은 실제 블로그 시스템이나 Notion API를 붙이지 않습니다.

## 글 1

```text
title:
date:
summary:
tags:
```

## 글 2

```text
title:
date:
summary:
tags:
```

## 작성 예시

```text
title: 첫 배포를 하며 배운 것
date: 2026-05-19
summary: Astro와 GitHub Pages로 개인 홈페이지를 처음 배포하며 배운 점을 정리합니다.
tags: Astro, GitHub Pages, 회고
```

```text
title: AI에게 코딩을 맡길 때 필요한 맥락
date: 2026-05-19
summary: 좋은 프롬프트가 왜 필요한지, 계획과 검증을 어떻게 요청하는지 정리합니다.
tags: Codex, AI, 작업기록
```

## AI에게 보낼 때 붙일 문장

```text
이 글 목록은 2회차용 샘플 데이터야.
지금은 Notion API를 붙이지 말고, 나중에 Notion 데이터로 바꿀 수 있는 구조만 만들어줘.
token, secret, API key는 만들거나 코드에 넣지 마.
```
