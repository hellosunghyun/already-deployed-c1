# 01. 시작 전 체크

## 목표

실습을 시작할 준비가 되었는지 확인합니다.

## 왜 하는가

첫날 가장 흔한 문제는 코드 문제가 아니라 **앱 실행, 계정 로그인, 배터리, 도구 선택 문제**입니다.
이 단계에서 막히면 뒤의 모든 실습이 밀립니다.

## 체크박스

- [ ] 노트북 배터리가 충분하다.
- [ ] 충전기를 가지고 있다.
- [ ] 사용할 AI 도구를 정했다. `Codex / Claude Code / Antigravity`
- [ ] 선택한 AI 도구가 열린다.
- [ ] 선택한 AI 도구에 `안녕!`이라고 보냈고 응답이 왔다.
- [ ] [GitHub](https://github.com/)에 로그인되어 있다.
- [ ] 오늘 최종 목표가 [GitHub Pages](https://pages.github.com/)의 `github.io` 링크라는 것을 이해했다.

## 도구별 메모

### Codex

- [Codex](https://chatgpt.com/codex)를 엽니다.
- 앱에서 `안녕!`이라고 보내봅니다.
- 응답이 오면 준비 완료입니다.

### Claude Code

- [Claude Code](https://docs.claude.com/en/docs/claude-code)를 엽니다.
- 프로젝트를 열기 전이라도 간단한 응답이 오는지 확인합니다.

### Antigravity

- [Antigravity](https://codelabs.developers.google.com/getting-started-google-antigravity)를 엽니다.
- Agent / Browser / Task list 관련 화면을 확인합니다.

## Ready Gate

다음 단계로 넘어가기 전에 아래가 되어 있어야 합니다.

- AI 도구가 정상 응답합니다.
- GitHub에 로그인되어 있습니다.
- 오늘 배포 목표가 `github.io`라는 것을 알고 있습니다.

되어 있나요?

- [ ] 네, 다음으로 넘어갈 수 있습니다.
- [ ] 아직입니다. 힌트를 보겠습니다.

</details>
