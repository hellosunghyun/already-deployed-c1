# Slide 06. 코딩 에이전트 공통 루프

오늘 설명은 [Codex](https://chatgpt.com/codex) 기준입니다.

하지만 [Claude Code](https://docs.claude.com/en/docs/claude-code)나 [Antigravity](https://codelabs.developers.google.com/getting-started-google-antigravity)를 쓰는 분들도 같은 흐름으로 갑니다.

```text
목표 제시
→ 계획 요청
→ 승인
→ 실행
→ 검증
→ 피드백
```

---

## 가장 중요한 습관

> 바로 실행시키지 말고,  
> 먼저 계획을 설명하게 합니다.
