# Slide 02. 오늘의 성공 기준

오늘 성공 기준은 3단계입니다.

## 최상

- [Astro](https://astro.build/) 개인 홈페이지 생성
- 내 컴퓨터에서 확인
- [GitHub](https://github.com/) 저장소에 업로드
- [GitHub Pages](https://pages.github.com/) 배포
- `https://<username>.github.io` 링크 확인

## 표준

- Astro 프로젝트 생성
- `localhost`에서 화면 확인
- GitHub 저장소에 코드 업로드

## 최소

- 어디서 막혔는지 캡처
- 다음 질문을 만들 수 있음
