# Slide 05. 좋은 요청의 9요소

AI에게 코딩을 부탁할 때는 아래를 생각합니다.

1. **목표** — 무엇을 만들고 싶은가
2. **현재 상태** — 지금 어디까지 되었는가
3. **사용 도구** — Codex / Claude Code / Antigravity
4. **기술 선택** — Astro, GitHub Pages 등
5. **배포 목표** — `github.io`
6. **제약** — 하지 말아야 할 것
7. **완료 기준** — 무엇이 되면 성공인가
8. **검증 방법** — 브라우저, build, 로그
9. **에러 자료** — 로그, 스크린샷, URL
