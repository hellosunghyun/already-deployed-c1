# Slide 11. Git과 commit을 왜 써야 하나?

[Git](https://git-scm.com/)은 변경 기록을 남기는 도구입니다.

## commit이란?

변경 사항에 이름표를 붙여 저장하는 것입니다.

예:

```text
feat: 홈 화면 초안 추가
fix: GitHub Pages 배포 설정 수정
chore: 초기 프로젝트 생성
```

---

## AI와 작업할수록 commit이 중요합니다

AI는 빠르게 많이 바꿉니다.

그래서 자주 commit해야 합니다.

- 실수했을 때 돌아갈 수 있음
- 어떤 변경이 문제였는지 알 수 있음
- 작업 단위가 명확해짐
