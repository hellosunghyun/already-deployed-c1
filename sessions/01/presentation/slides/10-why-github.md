# Slide 10. 왜 GitHub가 필요한가?

[GitHub](https://github.com/)는 코드를 온라인에 저장하는 공간입니다.

오늘 GitHub는 두 가지 역할을 합니다.

1. 내 홈페이지 코드를 저장합니다.
2. [GitHub Pages](https://pages.github.com/)가 배포할 기준이 됩니다.

---

## 왜 내 컴퓨터만으로는 부족할까요?

내 컴퓨터에만 있는 파일은 다른 사람이 볼 수 없습니다.

GitHub에 올려야:

- 나중에 다시 수정할 수 있고
- 실수했을 때 돌아갈 수 있고
- GitHub Pages가 사이트로 배포할 수 있습니다.
