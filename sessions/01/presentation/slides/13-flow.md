# Slide 13. 오늘 만들 흐름

```text
내 요구사항
  ↓
AI 에이전트
  ↓
Astro 프로젝트
  ↓
localhost 확인
  ↓
GitHub 저장소
  ↓
GitHub Pages
  ↓
github.io 링크
```

---

## 더 쉽게 말하면

```text
만든다
→ 확인한다
→ 저장한다
→ 배포한다
→ 링크로 본다
```

---

## SwiftUI 앱과 다른 점

SwiftUI 앱은 보통 Simulator나 실제 기기에서 실행해 확인합니다.

오늘 만드는 웹사이트는 브라우저에서 확인합니다.

```text
SwiftUI 앱 확인 → Simulator / 기기
웹사이트 확인   → localhost / github.io
```
