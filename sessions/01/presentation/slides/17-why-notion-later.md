# Slide 17. Notion은 왜 나중에 쓰나요?

[Notion](https://www.notion.so/)은 나중에 글과 프로젝트를 관리하는 공간으로 씁니다.

하지만 오늘은 붙이지 않습니다.

## 이유

Notion을 붙이면 한꺼번에 많은 개념이 나옵니다.

- Notion database
- API
- token
- secret
- build-time fetch
- GitHub Actions

오늘은 먼저 `github.io` 링크를 만드는 흐름에 집중합니다.

---

## 4회차 구조 미리 보기

```text
Notion DB
  ↓
Notion API
  ↓
Astro pages
  ↓
GitHub Pages
```
