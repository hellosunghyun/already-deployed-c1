# Slide 14. Codex 데모 흐름

오늘 데모는 길게 하지 않습니다.

## 보여줄 것

1. 프로젝트 폴더 만들기
2. [Codex](https://chatgpt.com/codex)에서 폴더 열기
3. 환경 확인 요청하기
4. 필요한 도구가 없으면 설치 계획 받기
5. `AGENTS.md` 만들기
6. Astro 프로젝트 만들기
7. `localhost`에서 확인하기
8. `@browser` 또는 Browser 도구로 화면 확인하기
