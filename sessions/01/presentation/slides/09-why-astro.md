# Slide 09. 왜 Astro인가?

[Astro](https://astro.build/)는 콘텐츠 중심 웹사이트에 잘 맞습니다.

이번 세션에서 만드는 것은 복잡한 앱이 아닙니다.

- 나를 소개하는 Home / About
- 내가 한 일을 보여주는 Projects
- 글과 기록을 모으는 Posts

이런 구조는 Astro와 잘 맞습니다.

---

## Astro를 쓰는 이유

- 개인 홈페이지에 적당한 복잡도
- 블로그 / 프로젝트 페이지로 확장하기 쉬움
- 정적 사이트로 배포하기 좋음
- 처음엔 HTML에 가깝게 시작 가능
- 나중에 필요하면 React / Vue 등도 붙일 수 있음
