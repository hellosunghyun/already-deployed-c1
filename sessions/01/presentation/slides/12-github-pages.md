# Slide 12. GitHub Pages란?

[GitHub Pages](https://pages.github.com/)는 GitHub 저장소에 있는 웹사이트 파일을 인터넷 주소로 보여주는 기능입니다.

오늘 목표 주소는 보통 이렇게 생깁니다.

```text
https://<github-username>.github.io
```

---

## 사용자 사이트 저장소 이름

```text
<github-username>.github.io
```

예:

```text
hello123.github.io
```
