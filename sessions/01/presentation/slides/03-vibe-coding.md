# Slide 03. vibe coding 다시 정의하기

## vibe coding ≠ 알아서 해줘

좋은 vibe coding은 아래 흐름입니다.

1. 목표를 말한다
2. 맥락을 준다
3. 제약을 말한다
4. AI에게 계획을 요청한다
5. 내가 승인한다
6. AI가 실행한다
7. 브라우저와 로그로 검증한다
8. 필요하면 다시 요청한다

---

## 오늘 계속 반복할 문장

> AI에게 코딩을 맡기는 것은 책임을 넘기는 것이 아닙니다.  
> 반복 작업을 위임하고, 검증 책임을 더 명확히 갖는 것입니다.
